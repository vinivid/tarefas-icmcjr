Como está sendo utilizado docker, seria necessário ter um servidor. 

Depois é necessário pushar esta imagem para o docker hub. 

Após isso, dar pull nesta imagem no servidor que deseja hospeda-lo. 

Configurar as portas para acessar o servidor. 

Rodar o container com as portas abertas.

Opicionalmente você poderia configurar um reverse proxy como nginx e comprar um dominínio para seu servidor.
