## Formato padrão

```json
{
  "error": "Código",
  "message": "Descrição"
}
```

## Erros padrões

Não esta sendo seguido erros padrões para códigos de status como 401, 403, 404.

Por exemplo, 404 é utilizado quando um email para login não é encotrado, entretanto isto não representa de nenhuma forma um erro e sim um uso semântico dos códigos de status.
