# Testes e validação

Está sendo utilizado o Jest com o Jest-expo para realizar os testes.
Eles são encontrados na diretório src/__tests__/

Foram testados apenas os componetes que apresetam alguma interação
de usuário.

O funcionamento de componentes de dependências externas não foram testados. 

## Validação manual 

Use o próprio Expo para validar o funcionamento manual do app:

```bash
npm start
```

## Rodar os testes

```bash
npm run test
```
